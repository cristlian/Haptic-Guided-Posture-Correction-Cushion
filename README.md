# Haptic-Guided Posture Correction Cushion

Minimal repository for the interactive dashboard prototype and top-view cushion analysis assets.

Included in this repo:
- `dashboard.html` for the interactive concept dashboard
- `_analyze_topview.py` for the image-based region analysis helper
- `smart-cushion-top-view.png` as the source cushion layout image

Coursework PDFs, presentation decks, extracted intermediates, and local Copilot assets remain in the workspace but are intentionally excluded from version control.
